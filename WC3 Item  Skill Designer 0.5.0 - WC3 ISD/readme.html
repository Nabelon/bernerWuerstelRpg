WC3 Item & Skill Designer is a tool to create professional item and skill tooltips in seconds!
This tool offers many useful features for mapper.

WC3 Item & Skill Designer ist completely customizable.

For RPG-Creators/Mappers this tool offers a lot of nice features that are used by default (But you can remove them), like Item Quality/Rarity, Item Typ and Required Class for the item.

There are some calulators, like a armor (damage reduction) calculator, damage/dps calculator, effective hp calculator (good for balancing, and with a evasion calulator (stack - cool) and a exp calculator. Just try it out!

And thats not all. There is also a name generator! Generate names for your heros with just a few clicks. Fully customizable.

Woah still more.. There is a w3x import too! (Not done atm).
No need to open world-editor just to edit some items or skills. And maybe even more.
So far i got the import of most of the files done (stuff i need, and stuff i dont need, but maybe will use later) and everything is working. Just need to add the list/edit feature.
